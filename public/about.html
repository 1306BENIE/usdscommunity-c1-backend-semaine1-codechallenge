À propos de l'application

📋 Description

Cette application de gestion des clients est une solution simple et efficace développée avec Express.js. Elle permet de gérer une liste de clients avec des opérations CRUD basiques.

🛠️ Technologies utilisées

  • Backend: Node.js avec Express.js
  • Frontend: HTML5, CSS3, JavaScript vanilla
  • Stockage: Mémoire (tableau JavaScript)
  • API: REST JSON

🔌 API REST

L'application expose une API REST complète pour la gestion des clients :

GET /api/clients Récupère la liste complète des clients
POST /api/clients Ajoute un nouveau client
DELETE /api/clients/:id Supprime un client par son ID

🧪 Tests avec cURL

Vous pouvez tester l'API directement depuis la ligne de commande :

Récupérer tous les clients :

curl http://localhost:3000/api/clients

Ajouter un nouveau client :

curl -X POST -H "Content-Type: application/json" -d '{"name": "John Doe", "email": "john@example.com"}' http://localhost:3000/api/clients

Supprimer un client :

curl -X DELETE http://localhost:3000/api/clients/1

🚀 Démarrage

Pour démarrer l'application :

  1. Installer les dépendances : npm install
  2. Démarrer le serveur : npm start
  3. Ouvrir dans le navigateur : http://localhost:3000
